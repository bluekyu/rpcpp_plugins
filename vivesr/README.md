# ViveSR Plugin

## Requirements
### Vive SRWorks SDK
Vive SRWorks SDK: https://developer.vive.com/resources/

"FindViveSR.cmake" will find files at below directory hierarchy:
```
─ ViveSR
  ├ headers
  │ ├ ViveSR_API.h
  │ └ ViveSR_Enums.h
  ├ libs
  │ └ ViveSR_API.lib
  └ x64
    └ Release
      └ (DLL files ...)
```
